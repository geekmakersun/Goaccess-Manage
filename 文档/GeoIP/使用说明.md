# GeoIP 数据库更新脚本使用说明

## 📋 脚本信息

- **脚本位置**: `脚本/GeoIP/更新GeoLite2.sh`
- **适用环境**: Linux/Unix 系统（CentOS/Ubuntu/Debian 等）、Windows Git Bash
- **数据源**: [Loyalsoldier/geoip](https://github.com/Loyalsoldier/geoip/releases)

## 🎯 功能特性

### ✅ 自动获取最新版本
- 通过 GitHub API 获取最新 release tag
- 无需手动指定版本号

### ✅ SHA256 校验
- 下载 `.sha256sum` 校验文件
- 验证文件完整性，确保下载正确

### ✅ 原子更新策略
- 先下载到临时文件
- 验证 SHA256 校验和
- 成功后再替换原文件
- 自动备份旧版本

### ✅ 版本信息持久化
- 创建 `GeoIP.version` 文件记录版本信息
- 包含版本号、更新时间、文件大小等信息
- 自动更新，无需手动维护

### ✅ 自动清理机制
- 自动清理超过 7 天的备份文件
- 避免备份文件占用过多空间

## 📖 使用方法

### Linux 服务器上运行

```bash
# 进入脚本目录
cd /www/wwwroot/GoAccess-管理/脚本/GeoIP

# 更新所有数据库
./更新GeoLite2.sh

# 只更新 Country 数据库（国家/地区）
./更新GeoLite2.sh -c

# 只更新 ASN 数据库
./更新GeoLite2.sh -a

# 强制更新
./更新GeoLite2.sh -f

# 显示版本信息
./更新GeoLite2.sh -v

# 清理旧的备份文件
./更新GeoLite2.sh -C
```

### Windows 用户

#### 方式 1: 在 Linux 服务器上运行（推荐）
将脚本上传到 Linux 服务器后执行

#### 方式 2: 使用 WSL
```bash

wsl --install

# 在 WSL 中运行
cd /mnt/c/Users/sunbingchen/工作区/GoAccess-管理/GeoIP
./更新GeoLite2.sh
```

#### 方式 3: 使用 Git Bash
```bash
# 在 Git Bash 中运行
cd /c/Users/sunbingchen/工作区/GoAccess-管理/GeoIP
./更新GeoLite2.sh
```

## 🔄 工作流程

1. **检测操作系统** - 自动识别运行环境
2. **显示当前版本** - 启动时显示已安装的数据库版本
3. **获取最新版本** - 通过 GitHub API 获取最新 release tag
4. **下载数据库文件** - 从 GitHub Releases 下载 .mmdb 文件
5. **下载校验文件** - 下载 .sha256sum 校验文件
6. **验证 SHA256** - 校验文件完整性
7. **原子更新** - 先下载到临时文件，验证后替换
8. **自动备份** - 备份旧版本到带时间戳的文件
9. **更新版本文件** - 保存最新的版本信息到 `GeoIP.version`
10. **显示最新版本** - 更新完成后显示新版本信息
11. **清理备份** - 自动清理超过 7 天的备份文件

## 📊 版本文件格式

`GeoIP.version` 文件示例：

```bash
# GeoIP 数据库版本信息
# 此文件由更新脚本自动生成，请勿手动修改
# 数据源: https://github.com/Loyalsoldier/geoip/releases

# 版本信息
VERSION="202605210"
UPDATE_TIME="2026-05-22 10:30:00"

# Country.mmdb (国家/地区数据库)
CITY_FILE="Country.mmdb"
CITY_SIZE="72345678"

# Country-asn.mmdb (ASN 数据库)
ASN_FILE="Country-asn.mmdb"
ASN_SIZE="12345678"
```

## 📝 注意事项

1. **运行环境**: 脚本需要在 Linux/Unix 系统上运行，Windows 用户请使用 WSL 或 Git Bash
2. **权限要求**: 需要有写入 GeoIP 目录的权限
3. **磁盘空间**: 至少需要 100MB 可用空间
4. **网络连接**: 需要能够访问 GitHub
5. **自动备份**: 更新前会自动备份，备份文件保留 7 天

## 🔧 故障排除

### 问题 1: 无法运行脚本
**原因**: Windows PowerShell 不支持 bash 命令
**解决**: 在 Linux 服务器上运行，或使用 WSL/Git Bash

### 问题 2: 下载失败
**原因**: 网络问题或 GitHub 访问受限
**解决**: 检查网络连接，或使用代理

### 问题 3: SHA256 校验失败
**原因**: 下载文件不完整或被篡改
**解决**: 重新运行脚本下载

### 问题 4: 权限不足
**原因**: 没有写入权限
**解决**: 使用 root 用户或 sudo 运行脚本

### 问题 5: 磁盘空间不足
**原因**: 可用空间小于 100MB
**解决**: 清理不需要的文件，释放磁盘空间
